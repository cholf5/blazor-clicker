**Cookie Clicker** was created by Julien "Orteil" Thiennot and DashNet

This project is:

- **An educational fan reimplementation.** Built to learn Blazor WebAssembly
  and to demonstrate what modern .NET can produce for a UI-heavy idle game.
- **Non-commercial.** No advertising, no monetization, no donations, no
  Patreon, no crypto. The hosted build (if any) is free to play.
- **Unaffiliated.** This project has no relationship with Orteil, DashNet,
  or the commercial Steam version of Cookie Clicker.

## What this repository does not include

- **No original source code** from Cookie Clicker. The gameplay was
  reimplemented from scratch in C# / Blazor.
- **No original assets.** No sprites, no audio, no icons from the original
  game are shipped in this repository. All in-game icons are Unicode
  emoji rendered by the user's system font.
- **No original flavor text.** All building descriptions, upgrade
  descriptions, and achievement names in this repository were rewritten
  for this remake. Where earlier drafts of this codebase mirrored the
  original wording, that wording has been replaced — see git history
  for the audit.

## What this repository does reference

- **Game mechanics and numeric constants** (base costs, CPS values, tier
  thresholds, price growth factor `1.15ⁿ`, golden cookie effects and
  timings). Game mechanics and numeric values are generally not eligible
  for copyright protection in most jurisdictions, but they are still
  clearly *inspired by* the original design.
- **Building names** (Cursor, Grandma, Farm, Mine, Factory, Bank, Temple,
  Wizard Tower, Shipment, Alchemy Lab, Portal, Time Machine). These are
  short generic terms that identify functional categories; they are
  retained so that the project is recognizable as a Cookie Clicker
  reimplementation.
- **The name "Cookie Clicker"** appears in the project title, page title,
  and documentation, always qualified with words like *"fan remake"*,
  *"reimplementation"*, or *"unofficial"* to make the non-official status
  unambiguous.
